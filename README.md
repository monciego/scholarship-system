# Scholarship System

Clone and setup

1. ```
   git clone https://github.com/monciego/scholarship-system.git
   ```

2. ```
   composer install
   ```

3. ```
   npm install
   ```

4. ```
   cp .env.example .env
   ```

5. ```
   php artisan key:generate
   ```

6. ```
   php artisan migrate
   ```

    Run the server

7. ```
   php artisan serve
   ```

    Open another terminal and run

8. ```
   npm run dev
   ```

Go to link http://127.0.0.1:8000/
